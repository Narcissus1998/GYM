# 碳循环.xlsx 分析

## 模型定位

这个工作簿不是饮食记录表，而是一个单页碳循环计算器：

1. 输入人体参数
2. 估算每日消耗
3. 生成日 / 周宏量预算
4. 把周预算拆成高碳、中碳、低碳三类日模板
5. 按 7 天节奏输出每天热量与热量赤字

## 关键输入

- 身高 `C2`
- 体重 `C3`
- 年龄 `C4`
- 性别 `C5`
- 训练时间 `C7`
- 训练系数 `C8`
- 体质 `B13`

## 关键公式

- 基础代谢 `C6`
  - 男：`10*C3 + 6.25*C2 - 5*C4 + 5`
  - 女：`10*C3 + 6.25*C2 - 5*C4 - 161`
- 训练消耗 `C9 = C7 * C8`
- 每日消耗 `C10 = C6 + C9`

### 体质决定的标准摄入系数

- 现行规则
  - 内胚：`碳水 2 / 蛋白 1.2 / 脂肪 0.6`
  - 中胚：`碳水 2.5 / 蛋白 1.4 / 脂肪 0.7`
  - 外胚：`碳水 3 / 蛋白 1.6 / 脂肪 0.8`
- 日摄入量
  - `G3 = G2 * 体重`
  - `H3 = H2 * 体重`
  - `I3 = I2 * 体重`
- 周摄入量
  - `G4:I4 = G3:I3 * 7`

### 三类碳循环模板

- 高碳
  - 碳水：周碳水 `50% / 2`
  - 蛋白质：等于日蛋白
  - 脂肪：周脂肪 `20% / 2`
- 中碳
  - 碳水：周碳水 `35% / 3`
  - 蛋白质：等于日蛋白
  - 脂肪：周脂肪 `35% / 3`
- 低碳
  - 碳水：周碳水 `15% / 2`
  - 蛋白质：等于日蛋白
  - 脂肪：周脂肪 `45% / 2`

### 每日热量

- 热量换算系数：
  - 碳水 `4`
  - 蛋白质 `4`
  - 脂肪 `9`
- 每天摄入热量：
  - `碳水*4 + 蛋白*4 + 脂肪*9`
- 每天热量赤字：
  - `每日消耗 - 当天摄入热量`

## 当前默认 7 天安排

- 周一 中
- 周二 高
- 周三 低
- 周四 中
- 周五 高
- 周六 低
- 周日 中

## 发现的问题

### 1. 体质规则冲突

底部参考表 `A32:D34` 与实际公式不一致：

- 底部参考表
  - 内：`2 / 1.2 / 0.8`
  - 中：`2.5 / 1.2 / 0.9`
  - 外：`3 / 1.2 / 1.0`

### 2. 训练等级设计未完成

`A8:B8` 有“新手/女生、中手/爱好、老手/热爱”的下拉校验，但实际计算使用的是 `C8` 数字系数，说明原工作簿的训练等级规则没有真正接入计算链。
